# Phonebook SPA </br></br>
# 1-Install dependencies:</br>
npm install</br>
# 2-Run Server:</br>
npm run server - (This will run the server on port 3002)</br>
# 3-Run React App:</br>
npm start</br></br>

For now you can:</br>
-Add contact (Name + Number)</br>
-Delete contact</br>
-Update contact</br>
-Search for contacts</br>
